urlify

Urlify - Refine text so it makes a good ID.

To do this, we remove apostrophes, replace nonsafe characters with hyphens, remove extra hyphens, truncate, trim hyphens, and make lowercase.

urlify(text: String): String
Parameters
text (String) Any text. Usually pulled from the webpage element we are linking to.
Returns
String: hyphen-delimited text for use in IDs and URLs.
