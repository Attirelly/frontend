      ListingFooter component

      The main footer component for the website. It provides company contact information, social media links, and key navigation links for sellers and company policies.

      Features

      • Renders a responsive, multi-column layout that stacks on mobile devices.
      • Company Information Column:
      • Displays the company logo/brand name.
      • Shows physical address, phone number, and a clickable email link.
      • Dynamically renders social media icons with links to company profiles.
      • Selling Store Column: Provides navigation links for sellers (Sign In, Sign Up, Ambassador Program).
      • Company Column: Contains links to legal pages like Privacy Policy and Terms & Conditions.
      • Includes a horizontal divider for clean visual separation from the page content above.

      Logic Flow

      This is a stateless, presentational component with no internal logic or data fetching.

      • The social media links are dynamically generated by mapping over the socialPlatforms constant array.
      • Each link is constructed using the href, name, and hoverColor properties from the array.
      • Navigation is handled by the Next.js Link component for client-side routing.

      Key Data Structures

      • socialPlatforms: A local constant array of objects. Each object defines a social media platform with its name (used for the icon filename), href (the external URL), and hoverColor (a Tailwind CSS class for styling).

      Imports

      • Core/Libraries: React from 'react'; Image, Link from Next.js.
      • Utilities: manrope, roboto, rosario from @/font for consistent typography.

      Returns Element

      The rendered footer component.
